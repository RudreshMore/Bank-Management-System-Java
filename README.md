# Bank-Management-System-Java
A simple Java console-based Bank Management System using OOP &amp; ArrayList.

Bank Management System (Java)
A simple console-based Bank Management System created in Java using:
Object-Oriented Programming
ArrayList
CRUD operations
Scanner input

 Features:
✔ Create Account
✔ View All Accounts
✔ Deposit Money
✔ Withdraw Money
✔ Check Balance
✔ Delete Account
✔ Fully menu-driven


Technologies Used:
Java 8+
OOP Principles
Collections Framework

How to Run :
javac src/in/rudresh/*.java
java src/in/rudresh/BankApp

Output (Sample):
===== BANK MENU =====
1. Create Account
2. View All Accounts
3. Deposit Money
4. Withdraw Money
5. Check Balance
6. Delete Account
7. Exit
